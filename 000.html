Installing/Updating Exercises on your VM:

The set of CCSO Exercises hosted here on our website is designed to be easy to install and update. Users can run the command on this page on an Ubuntu Virtual Machine to access the challenges. Users can also visit individual challenge pages to pull single challenges down, but this feature is to be removed once testing this setup is complete. While the challenges are being tested, a list of all challenge passwords is provided upon install. When the challenges exit beta, the password for the next "level" will be provided as a "flag" upon completion of the current "level". This is similar to OverTheWire's "Bandit" challenges. What differentiates the CCSO challenge set is the specificity of our challenges to the skills we would like to help our members build. We can also show more advanced members how the challenges are constructed, so that they might gain a deeper understanding of challenge design, and so that they might help develop new challenges.

On a new Ubuntu VM (or existing VM of which you have a clean snapshot), run the following:

curl -L https://git.io/fx7XO | sudo bash
